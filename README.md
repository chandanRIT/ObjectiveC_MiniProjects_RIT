# ObjectiveC_MiniProjects_RIT

Two mini-projects in Objective C: A simple Cocoa app and an iPhone app. Both the projects follow Model-View-Control design paradigm.

### Menu-File Format:

a. File to be used should be in comma separated and each line contains individual entry

b. Please see menu.txt for the exact format

### Usage of the system: 

a. File is loaded automatically on loading the app.

b. One should see the menu loaded in the main screen ('iMenu/CocoaMen') in a table if the file format is as specified and  loaded successfully. Else an error message will be displayed in the table.

c. Choose items by changing the values of the quantity by tapping on a row respective row. All the non-zero quantity items are added to the order.

d. Press the "Get Bill" button to compute the bill and it will be displayed in the next screen titled 'Bill'.

e. incorrect file formats are handled using try catch mechanism.

f. "Clear Bill" button is used to reset the quantities and restore the menu to place a  new order.
